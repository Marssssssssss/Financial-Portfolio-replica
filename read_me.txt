Il PPT è salvato sia in PDF che in formato PPT, il pdf è più scorrevole, però si consiglia il PowerPoint poiché è presente un'animazione, che si può trovare anche nel file MatLab per i pesi del Kalman Filter.

Si consiglia la lettura nel seguente ordine:
1. Data Exploration - file Python (Jupiter), una semplice esplorazione dati;
2. Regression_all - file sia html (per una lettura veloce) sia per una visualizzazione migliore in R Notebook, sono presenti tutte le regressioni con spiegazioni molto dettagliate che approfondiscono le scelte dei parametri molto più di quanto fatto sul PowerPoint;
3. Kalman Filter - file MatLab Live Script, anche qui c'è un approfondimento rispetto al solo PowerPoint. ATTENZIONE file .mat della replica necessario;
4. Interface - file Python, non abbiamo ritenuto fosse necessario un notebook in quanto bisogna solo far runnare per far comparire l'interfaccia, si consiglia di installare precedentemente le librerie.
